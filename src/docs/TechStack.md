# Project Synapse: Tech Stack

## Frontend
* **Framework**: React 19 (Vite + TypeScript).
* **Styling**: Tailwind CSS (Custom Neon Theme).
* **Animations**: GSAP (for mouse followers) and Framer Motion (for page transitions).
* **Components**: Reactbits (Scroll Stack), Lightwind UI (3D Carousel), and Lucide React (Icons).

## Backend (Planned)
* **API Architecture**: Integrated by Sanyam.
* **Data Sources**: GitHub API, Discord Webhooks, and internal AI analysis engines.
